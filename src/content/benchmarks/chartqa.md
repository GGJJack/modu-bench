# ChartQA (차트 이해)

## 개요
ChartQA는 차트를 기반으로 복잡한 시각적 논리 및 산술 연산 능력을 평가하기 위한 대규모 벤치마크입니다. 기존 데이터셋들이 템플릿 기반의 단순 질문에 머물렀던 것과 달리, 여러 차트의 시각적 특징과 데이터 테이블을 통합적으로 다뤄야 풀 수 있는 복잡한 추론 문제들에 중점을 둡니다.

## 평가 데이터 구성
인간이 직접 작성한 9,600여 개의 질문과, 차트 요약을 바탕으로 생성된 23,100여 개의 질문이 포함되어 있습니다.
