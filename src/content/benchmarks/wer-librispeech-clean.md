# WER (LibriSpeech Clean)

## 개요
LibriSpeech clean 테스트셋 기준 단어 오류율(Word Error Rate)을 측정하는 벤치마크.

## 평가 방법
음성 인식 모델이 예측한 텍스트와 실제 텍스트 간의 단어 오류율을 계산.
