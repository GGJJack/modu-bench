# Humanity's Last Exam (HLE) (with tools)

## 개요
Humanity's Last Exam (HLE)의 도구 사용 허용(with tools) 버전은 최첨단 AI 모델들이 외부 도구(예: 코드 실행기, 웹 브라우저, 검색 엔진 등)를 활용하여 문제를 해결하는 능력을 평가하는 학술 벤치마크입니다. 현존하는 최고 난이도의 폐쇄형 질문들로 구성되어 있습니다.

## 평가 방법
기본적인 문제 세트는 HLE와 동일하나, 모델이 외부 도구를 호출하여 추가적인 계산을 수행하거나 정보를 검색할 수 있도록 허용된 환경에서 진행됩니다. 전 세계의 주제별 전문가들에 의해 개발된 2,500개의 질문(수학, 인문학, 자연 과학 등)을 해결하는 과정에서 도구 사용의 효율성과 정확성을 함께 측정합니다.

## 점수 해석
도구를 사용할 수 있음에도 불구하고, HLE 벤치마크의 질문들은 단순 검색만으로는 해결하기 어려운 깊이 있는 추론을 요구하므로 여전히 낮은 정확도를 보일 수 있습니다. 이 평가에서의 높은 점수는 모델이 복잡한 논리적 추론 능력과 더불어 실세계 도구를 적절히 조합하여 사용하는 능력이 뛰어남을 나타냅니다.
