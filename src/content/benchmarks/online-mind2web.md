# Online-Mind2Web

## 개요
Online-Mind2Web은 웹에서 언어 지시문을 따라 복잡한 작업을 수행할 수 있는 범용 웹 에이전트를 개발하고 평가하기 위한 벤치마크입니다. 137개 웹사이트와 31개 도메인에 걸쳐 2,350개의 태스크를 포함하고 있으며, 시뮬레이션 환경이 아닌 실제 웹사이트를 기반으로 하여 현실적인 환경에서의 에이전트 성능을 측정합니다.

## 평가 방법
기존 웹 에이전트 벤치마크가 시뮬레이션 환경이거나 제한된 태스크만 제공했던 것과 달리, Mind2Web은 31개 도메인의 실제 웹사이트를 대상으로 교차 작업, 교차 웹사이트, 교차 도메인 일반화 능력을 폭넓게 평가합니다. 모델은 Click, Hover, Type, Select 등의 작업을 수행하며 평가 단위는 %를 사용합니다.

## 점수 해석
이 벤치마크의 점수는 다양한 웹 환경에서의 범용적인 태스크 일반화 능력과 실제 웹사이트에서의 행동 수행 능력을 의미합니다.

## 한계와 비판
실제 웹 환경은 수시로 변경되므로 벤치마크 데이터와 실제 웹 환경 간의 불일치가 발생할 수 있으며, raw HTML이 너무 커서 모델 처리 효율성에 어려움이 있을 수 있습니다.
