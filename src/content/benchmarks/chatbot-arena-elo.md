# Chatbot Arena ELO

## 개요
Chatbot Arena는 인간의 선호도를 기반으로 대형 언어 모델(LLM)을 평가하기 위한 오픈 플랫폼입니다. 기존의 정적 벤치마크들이 실제 인간 사용자의 복잡하고 다양한 선호도를 완벽히 반영하기 어렵다는 한계를 극복하기 위해 설계되었습니다. 사용자들은 익명의 두 모델과 상호작용하고 어느 모델의 답변이 더 우수한지 투표합니다.

## 평가 방법
이 벤치마크는 크라우드소싱을 통한 쌍대 비교(Pairwise comparison) 방식을 사용합니다.
1. 사용자는 플랫폼에서 프롬프트를 입력합니다.
2. 두 개의 익명 모델이 각각 답변을 생성합니다.
3. 사용자는 두 답변 중 더 나은 것을 선택하거나, 무승부(tie), 혹은 둘 다 나쁨(both are bad)을 선택하여 투표합니다.
4. 수집된 투표 데이터는 체스 등에서 널리 쓰이는 ELO 레이팅 시스템을 통해 분석되어, 각 모델의 상대적인 순위와 점수가 산출됩니다.

## 점수 해석
ELO 점수는 모델 간의 상대적인 성능을 나타냅니다. ELO 점수가 높을수록 사용자들이 해당 모델의 답변을 더 선호함을 의미합니다. ELO 시스템의 특성상 점수 차이를 통해 한 모델이 다른 모델을 이길 확률을 통계적으로 추정할 수 있습니다.

## 특징과 의의
* **실제 사용 사례 반영**: 고정된 프롬프트 데이터셋이 아닌, 사용자들이 실시간으로 입력하는 다양한 프롬프트를 처리하므로 모델의 실제 활용 능력을 더 잘 반영합니다.
* **커뮤니티 주도**: 크라우드소싱을 통해 광범위하고 다양한 평가 데이터를 수집하여 특정 편향을 줄일 수 있습니다.
* **동적 랭킹**: 새로운 모델이 지속적으로 추가되고 사용자 투표가 계속되므로, 리더보드가 실시간에 가깝게 업데이트됩니다.
