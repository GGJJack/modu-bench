## 개요
GPT-4o mini는 OpenAI가 2024년 7월 18일에 발표한 가장 비용 효율적인 고성능 소형 멀티모달 모델입니다. 인공지능 기술의 대중화와 비용 절감을 목표로 설계된 이 모델은 기존의 주력 소형 모델이었던 GPT-3.5 Turbo를 완전히 대체하며, 훨씬 더 저렴한 가격으로 프론티어급 모델에 준하는 추론 성능을 선사합니다. 텍스트와 시각(Vision) 정보를 동시에 이해하는 네이티브 멀티모달 아키텍처를 취하고 있으며, 매우 빠른 응답 지연 시간(Low Latency)을 기반으로 실시간 대화 및 워크플로우 자동화 애플리케이션 개발에 최적화되어 있습니다.

## 기술 특징
* **비용 및 토크나이저 최적화**: 100만 입력 토큰당 15센트($0.15), 100만 출력 토큰당 60센트($0.60)라는 파격적인 요금으로 책정되어 기존 GPT-3.5 Turbo 대비 60% 이상 저렴합니다. 또한 GPT-4o와 동일한 개선된 토크나이저를 공유하여 한국어를 비롯한 비영어권 텍스트 처리의 토큰 효율성이 크게 향상되었습니다.
* **128K 컨텍스트 및 강력한 스펙**: 소형 모델임에도 128,000 토큰의 광활한 컨텍스트 윈도우를 기본 지원하며, 단일 요청당 최대 16,000 토큰에 달하는 방대한 텍스트 출력이 가능합니다. 지식 컷오프는 2023년 10월까지의 데이터를 반영하고 있습니다.
* **학술적 성능 우위**: MMLU(텍스트 이해 및 추론) 벤치마크에서 82.0%를 기록하여 Gemini Flash(77.9%) 및 Claude Haiku(73.8%) 등 경쟁사 소형 모델들을 큰 격차로 압도했습니다. 수학적 추론을 측정하는 MGSM에서 87.0%, 코드 생성을 평가하는 HumanEval에서 87.2%, 그리고 MMMU 멀티모달 추론 평가에서 59.4%를 기록하는 등 다방면에서 정량적 성능 우위를 검증받았습니다.
* **Instruction Hierarchy 보안 기술**: API 환경에서 작동하는 모델 중 최초로 '명령 계층 구조(Instruction Hierarchy)' 방법론이 기본 내장되었습니다. 이 기술은 시스템 프롬프트나 개발자의 지시를 최우선 순위로 간주하고 사용자 입력에 숨겨진 악의적 명령을 하위로 취급함으로써, 탈옥(Jailbreak) 및 프롬프트 주입(Prompt Injection) 시도를 강력히 무력화합니다.

## 사용 사례
* **대량 다중 API 체이닝**: 극히 저렴한 호출 단가와 빠른 응답성 덕분에, 하나의 사용자 질의를 처리하기 위해 내부적으로 여러 번의 모델 호출이나 병렬 처리를 반복 수행하는 고성능 자율형 AI 에이전트 서비스 개발에 적극적으로 도입됩니다.
* ** receipt 및 영수증 데이터 추출**: OpenAI의 얼리 파트너사인 Ramp 등의 실제 적용 사례에 따르면, GPT-4o mini는 영수증이나 청구서 이미지 파일로부터 필요한 구조화된 텍스트 필드를 정확히 발췌해내는 비정형 정보 추출 업무에서 구형 모델 대비 탁월한 가성비를 입증했습니다.
* **이메일 초안 자동 생성**: Superhuman과 같은 이메일 생산성 도구에서는 긴 대화 맥락과 히스토리를 완벽하게 분석하여 전문적인 맞춤형 회신 메일 초안을 지체 없이 빠르게 제안하는 비서 기능에 활용하고 있습니다.

## 한계
* **완벽한 멀티모달 출력 지원의 유예**: 출시 초기에는 API 환경에서 오직 텍스트 입력과 비주얼 이미지 입력(Vision)만을 공식 지원합니다. 비디오 및 네이티브 오디오의 양방향 완전 입력과 실시간 출력(Speech-to-Speech 등)은 점진적인 업데이트를 거쳐 단계적으로 제공될 예정입니다.
* **특정 미세 조정의 자원 한계**: 초경량 구조를 취하고 있어 일상적이고 표준적인 유스케이스에서는 매우 뛰어난 성능을 발휘하나, 극도로 복잡하고 특수한 전문 지식을 요구하는 도메인의 경우 플래그십 등급 모델인 GPT-4o나 o1 라인업에 비해 디테일한 추론 깊이와 논리 전개 면에서 다소 밀리는 경향을 보입니다.
