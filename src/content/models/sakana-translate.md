# Sakana Translate 소개

## 개요
Sakana Translate는 Sakana AI가 2026년 7월 6일에 발표한 번역 및 언어 지능 서비스입니다. 이 서비스는 해외의 프론티어 모델을 일본 사양에 최적화시킨 'Namazu' 모델 시리즈를 엔진으로 채택하였습니다. 단순히 단어를 치환하는 번역을 넘어, 일본어, 영어, 중국어 간의 문맥과 톤, 사회적 거리감까지 반영하는 '깊은 번역(Deep Translation)'을 지향합니다. 누구나 웹 앱을 통해 무료로 사용할 수 있도록 공개되었습니다.

## 기술 특징
Sakana Translate는 일본어의 언어적 특성 및 문화를 깊이 이해하도록 설계된 사후 학습(Post-training) 기술의 결집체입니다. 비즈니스 이메일의 정중한 표현이나 인터넷 슬랭, 문화적 고유 명사 등 기존 글로벌 번역 툴이 놓치기 쉬운 세밀한 뉘앙스를 정확하게 포착합니다. 정량적 평가인 WMT 2024 General Translation 태스크의 XCOMET-XL 지표에서도 글로벌 최정상급 모델들과 대등한 수준의 점수를 기록하며 실용적인 번역 품질을 증명하였습니다.

## 사용 사례
이 모델은 세 가지 주요 모드를 통해 다양한 언어 작업을 지원합니다.
- **번역 모드**: 최대 5,000자까지의 장문을 실시간 스트리밍으로 번역하며, 비즈니스 문서나 웹 페이지 번역에 적합합니다.
- **첨삭(Proofread) 모드**: 입력한 문장을 더 자연스럽고 적절한 표현으로 다듬어주며, 변경 사항을 하이라이트로 표시하여 비즈니스 이메일 작성 시 품질 체크에 유용합니다.
- **질의(Ask) 모드**: 번역이나 첨삭 결과에 대해 추가적인 질문을 던져 단어 선택의 이유나 미묘한 뉘앙스 차이를 해설받을 수 있어 어학 학습 도구로도 활용 가능합니다.

## 한계
현재 Sakana Translate는 웹 기반 서비스로 제공되며, API 제공이나 엔터프라이즈 기능(SSO, 감사 로그 등)은 향후 로드맵에 포함되어 있습니다. 또한 텍스트 기반 번역에 집중하고 있어, 이미지나 음성이 포함된 멀티모달 번역 기능은 지원하지 않습니다.
