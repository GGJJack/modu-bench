# Command A+ 소개

## 개요
Command A+는 Cohere가 2026년 5월 19일에 발표한 첫 번째 희소 혼합 전문가(Sparse Mixture-of-Experts, MoE) 아키텍처 기반의 대형 언어 모델입니다. 이 모델은 복잡한 추론, 멀티모달 입력 처리, 그리고 다국어 에이전트 작업을 위해 구축된 기업용 '워크호스(Workhorse)'를 지향합니다. 특히 이전 세대인 Command A 제품군의 기능을 단일 모델로 통합하였으며, Apache 2.0 라이선스로 공개되어 개발자들이 자율적으로 배포하고 제어할 수 있는 주권적 AI(Sovereign AI)를 실현하는 데 중점을 두었습니다.

## 기술 특징
Command A+는 총 2,180억 개의 파라미터를 보유하고 있으나, 추론 시에는 토큰당 250억 개의 파라미터만 활성화되는 MoE 구조를 사용하여 높은 효율성을 달성했습니다. 128k의 입력 컨텍스트와 64k의 최대 생성 길이를 지원하며, 텍스트와 이미지 입력을 동시에 처리할 수 있는 네이티브 멀티모달 능력을 갖추고 있습니다. 특히 W4A4(4비트 가중치 및 활성 함수) 양자화 기술과 증류(Distillation) 기법을 결합하여 성능 저하를 최소화하면서도 단 2장의 NVIDIA H100 또는 1장의 B200 GPU에서 구동이 가능하도록 설계되었습니다. 또한, 한국어를 포함한 48개 언어를 지원하며, 새로운 토크나이저를 통해 한국어 기준 약 16%의 토큰 압축 효율 개선을 이루었습니다.

## 사용 사례
이 모델은 복잡한 도구 사용(Tool Use)과 다단계 추론이 필요한 에이전트 워크플로우에 최적화되어 있습니다. Cohere의 기업용 플랫폼인 'North'와의 연동을 통해 에이전틱 질의응답, 스프레드시트 데이터 분석, 장기 기억 기반의 연속적인 대화 작업 등에서 뛰어난 성능을 보여줍니다. 또한, 모델 응답 내에 정보의 출처를 명시하는 인용(Citations) 기능과 생각의 과정을 투명하게 보여주는 추론 태그(<START_THINKING>) 기능을 지원하여 기업 환경에서 요구되는 답변의 신뢰성과 투명성을 높였습니다.

## 한계
Wiki 및 공식 기술 블로그에 따르면, Command A+는 MoE 아키텍처의 특성상 전체 모델 가중치를 저장하기 위한 큰 메모리 공간이 필요합니다. 비록 W4A4 양자화를 통해 구동 장벽을 낮추었으나, 최적의 성능과 레이턴시를 확보하기 위해서는 특정 하드웨어(B200 또는 H100급)와 vLLM 등의 특정 추론 프레임워크 사용이 권장됩니다. 또한 멀티모달 입력은 지원하지만 출력은 현재 텍스트와 추론 결과에 한정되어 있습니다.
