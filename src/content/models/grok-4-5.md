# Grok 4.5 소개

## 개요
Grok 4.5는 2026년 7월 8일 xAI(SpaceXAI)가 발표한 최신 플래그십 대형 언어 모델(LLM)입니다. 코딩, 에이전트 작업, 그리고 고도의 지식 작업(Knowledge work)을 수행하는 데 있어 역대 가장 뛰어난 지능을 갖추도록 설계되었습니다. 특히 AI 기반 코드 에디터인 Cursor와 협력하여 훈련되었으며, 실제 엔지니어링 환경에서의 문제 해결 능력을 극대화한 모델입니다.

## 기술 특징
Grok 4.5는 수만 개의 NVIDIA GB300 GPU 클러스터에서 훈련되었으며, 대규모 훈련의 안정성을 위한 최신 기술들이 적용되었습니다. 단순히 데이터의 양을 늘리는 것에 그치지 않고, 고도로 필터링된 데이터 큐레이션과 다단계 소프트웨어 엔지니어링 작업에 초점을 맞춘 강화 학습(RL)을 통해 '토큰당 지능'을 비약적으로 높였습니다.

성능 면에서는 80 TPS(Tokens Per Second)의 빠른 추론 속도를 제공하면서도, 경쟁 모델 대비 압도적인 토큰 효율성을 자랑합니다. 예를 들어, SWE Bench Pro 과제 해결 시 Claude 3.5 Opus 4.8(Max) 대비 약 4.2배 적은 토큰(평균 15,954개)만으로도 과제를 완수할 수 있습니다. 또한 고도로 비동기화된 훈련 스택을 통해 에이전트가 수 시간 동안 자율적으로 동작하며 학습할 수 있는 환경을 구축했습니다.

## 주요 활용 사례
Grok 4.5는 복잡한 Rust나 C/C++ 코딩부터 엔드-투-엔드 앱 빌딩까지 광범위한 엔지니어링 작업에 최적화되어 있습니다. 또한 'Grok Build'의 기본 모델로서 엑셀(Excel)의 복합 수식 및 다중 시트 모델 구축, 파워포인트(PowerPoint)의 네이티브 도형을 활용한 다이어그램 설계, 워드(Word)의 명확한 문장 작성 등 오피스 생산성 도구와의 유기적인 통합 기능을 제공합니다.

## 한계
Grok 4.5는 현재 미국을 포함한 대부분의 지역에서 API 및 서비스를 통해 제공되고 있으나, 유럽연합(EU) 지역은 규제 준수 절차로 인해 2026년 7월 중순 이후부터 순차적으로 지원될 예정입니다. 또한 에이전트 능력이 강화되었음에도 불구하고, 수 시간에 걸친 장기 작업 시에는 여전히 인간의 감독과 정밀한 검토가 권장됩니다.
