# HCX-DASH-002 소개

## 개요
HCX-DASH-002는 네이버클라우드(NAVER Cloud)의 초대규모 AI인 하이퍼클로바X(HyperCLOVA X) 제품군 중 가장 높은 효율성을 자랑하는 경량화 엔진입니다. 기존의 플래그십 모델인 HCX-007이 심층적인 사고와 복잡한 문제 해결에 최적화되어 있다면, HCX-DASH-002는 빠른 응답 속도와 저렴한 비용을 바탕으로 실시간 서비스 및 대량의 데이터 처리에 특화되어 있습니다. 한국어 특화 성능을 유지하면서도 운영 효율을 극대화하려는 기업 사용자들에게 최적의 선택지를 제공합니다.

## 기술 특징
HCX-DASH-002는 32,000 토큰의 컨텍스트 윈도우를 지원하여, 비교적 긴 문맥을 유지하면서도 빠른 추론 성능을 보장합니다. 특히 이 모델은 '튜닝(Fine-tuning)' 기능을 지원하여, 기업이 보유한 자체 데이터를 학습시켜 특정 도메인이나 업무 스타일에 최적화된 맞춤형 모델을 생성할 수 있습니다. 또한 Function calling 기능을 탑재하고 있어 외부 API나 함수와의 동적인 상호작용이 가능하며, 이를 통해 실시간 정보를 반영한 답변 생성이 가능합니다.

## 사용 사례
이 모델은 빠른 반응 속도가 중요한 챗봇 시스템, 대량의 문서를 신속하게 분류하거나 요약하는 작업, 그리고 단순 반복적인 텍스트 생성 업무에 널리 활용됩니다. 특히 고객 상담 챗봇이나 사내 지식 검색 시스템에서 비용 부담을 줄이면서도 안정적인 한국어 성능을 확보하고자 할 때 주로 선택됩니다. 또한 튜닝 기능을 활용하여 특정 브랜드의 어투를 학습시키거나 전문 용어가 빈번한 기술 문서를 처리하는 전용 모델로 확장하여 사용할 수 있습니다.

## 한계
경량화된 모델의 특성상 고도의 수리적 추론, 복잡한 논리 구성, 혹은 매우 긴 호흡의 창의적 글쓰기에서는 상위 모델인 HCX-007 대비 정교함이 다소 부족할 수 있습니다. 또한 128,000 토큰을 지원하는 상위 모델들과 달리 32,000 토큰의 제한된 컨텍스트 윈도우를 가지고 있으므로, 초거대 리포지토리 분석이나 극도로 긴 대화 이력 유지에는 한계가 있을 수 있습니다.
