# Composer 2.5 소개

## 개요
Composer 2.5는 xAI가 2026년 6월에 공개한 코딩 및 시스템 구축 특화 모델입니다. xAI의 개발 도구인 'Grok Build' 환경에 최적화되어 설계되었으며, 기존 모델들보다 더 복잡한 개발 지시문을 정확하게 수행하는 데 중점을 두었습니다. 사용자는 Grok Build 내에서 `/model` 명령어를 통해 손쉽게 이 모델로 전환하여 작업을 수행할 수 있습니다.

## 기술적 특징
Composer 2.5의 가장 큰 강점은 '장기 실행 작업(long-running tasks)'에서의 안정성과 효율성입니다. 대규모 코드베이스를 분석하거나 여러 단계에 걸친 복잡한 리팩토링 작업을 수행할 때 지시문 이행 능력이 탁월한 것으로 평가받습니다. 또한 xAI의 최신 아키텍처(Grok 4 기반)를 활용하여 빠른 추론 속도를 유지하면서도 높은 수준의 코딩 지능을 제공합니다.

## 사용 및 접근성
이 모델은 현재 xAI의 CLI 도구와 Grok Build 환경을 통해 제공됩니다. SuperGrok 및 X Premium+ 구독자들은 추가 비용 없이 사용할 수 있으며, 개발자들은 API 콘솔을 통해 자신의 워크플로우에 통합할 수 있습니다. xAI는 이 모델이 단순한 코드 작성을 넘어 시스템 아키텍처 설계와 복잡한 디버깅 세션에서 개발자들의 핵심 조력자가 될 것으로 기대하고 있습니다.
